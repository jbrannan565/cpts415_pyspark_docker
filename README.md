## Dependencies
[docker](https://docs.docker.com/get-docker/)

### installing docker on ubuntu
```bash
./install_docker.sh
```

## Getting Started
1. Move into repo
```bash
cd cpts415_pyspark_docker
```
2. Run setup
```
./start.sh
```
3. Follow on screen instructions
